Knox Recovery UI Walkthrough

Step 1: Download and Install
  1. Download the Knox installer from Veracity Downloads.
  2. Run the installer:
    sudo ./install-knox.sh
Step 2: Configure Settings
FieldDescriptionDefault Value
PortPort Knox listens on9443
Data DirectoryLocation for backup metadata/var/knox
Enable LogsWhether to enable debug loggingtrue
Step 3: Validate and Monitor
  • Access the Knox dashboard at https://knox.veracity.io.
  • Check job status, logs, and alerts.
  • Export logs for troubleshooting.
